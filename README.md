# Airsoft Equipment Tracker

A modern Blazor web application to manage and track airsoft equipment, including replicas, gear, and loadouts.

## 🚀 Features
- Add and manage equipment items
- Categorize equipment (Primary, Secondary, Gear, etc.)
- Filter equipment dynamically
- Dashboard with statistics (in progress)

## 🛠️ Tech Stack
- Blazor (.NET)
- C#
- HTML / CSS
- (Later: database integration)

## 🎯 Goal
This project is part of my portfolio and focuses on:
- Building a structured Blazor application
- Practicing clean architecture (models, services, components)
- Creating a user-friendly interface for managing gear

## 🧠 What I'm Learning
- Working with Blazor components
- Data binding and state management
- UI structuring and reusable components
